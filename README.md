### 2.変数 $age に年齢を代入

### 3.変数 $hobbies に趣味を配列で3つ以上代入

> 「こんにちは、[名前]です」
> 「年齢は[年齢]歳です」
> 「趣味：」の後に趣味をリスト形式で表示

## 問題2
